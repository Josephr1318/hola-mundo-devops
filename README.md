# Aplicación Web Hola Mundo - Práctica DevOps

Esta es una aplicación web simple creada para practicar el ciclo completo de DevOps.

## Características

- Interfaz web responsiva con HTML, CSS y JavaScript
- Mensajes de "Hola Mundo" en múltiples idiomas
- Diseño moderno y atractivo
- Preparada para ser contenedorizada con Docker

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Docker
- Nginx (servidor web en el contenedor)

## Instrucciones para construir y ejecutar con Docker

### 1. Construir la imagen Docker

docker build -t hola-mundo-devops .
